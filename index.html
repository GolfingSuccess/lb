APL language bar for most websites and browsers

This provides a means to easily enter APL characters into pretty much any input field using a language bar as on this page.

Try it here:

Desktop browser installation

  1. Drag this link to your bookmarks bar: lb

Mobile browser installation

  1. Add a bookmark for this page.
  2. Edit the bookmark's target URL address to javascript:(d=>{let%20e=d.createElement("script");e.src="https://abrudz.github.io/lb/lb.js";d.body.appendChild(e)})(document)

Usage

  1. Navigate to the web page where you want to enter APL characters.
  2. Click on the lb bookmark. A grey language bar should appear at the web page's top.
  3. You can use any of the following methods to enter APL characters:
    1. Use backtick ` as prefix APL key, e.g. `a gives ⍺.
    2. Enter a combo sequence and hit tab, e.g. aaTab gives ⍺.
    3. Click on symbols in the language bar.

Hover over a symbol in the language bar for keybindings and combo sequences.
